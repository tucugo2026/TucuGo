# Arquitectura inicial TucuGo

## Módulos

### Pasajero
- login y registro
- solicitar viaje
- ver historial

### Conductor
- registro como conductor
- carga de documentación
- estado pendiente/aprobado
- viajes

### Admin
- revisar choferes
- aprobar o rechazar
- ver viajes
- ajustar tarifas

## Firebase

### Colecciones sugeridas
- usuarios
- conductores
- vehiculos
- viajes
- pagos
- mensajes
- configuracion

## Flujo de conductor
1. Se registra
2. Completa formulario
3. Sube documentos
4. Queda pendiente
5. Admin aprueba
